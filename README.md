# RK Cron Monitor

Free API to monitor cron jobs. Detect missed runs, false alerts, and calculate downtime.

**Endpoints:**
- `POST /api/check` - Monitor job health with F1+F3+F4 logic
- `POST /api/parse` - Convert cron expressions to human readable

Built for devs who are tired of paying for cron monitoring.
